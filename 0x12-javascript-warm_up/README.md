# JavaScript Warm-Up

This repository contains a collection of JavaScript scripts that cover various basic and advanced concepts in JavaScript programming. Each script demonstrates a specific concept or functionality.

## Getting Started

To run these scripts, you need to have Node.js installed on your system. If you don't have Node.js, you can download it from [nodejs.org](https://nodejs.org/).

### Clone the Repository

You can clone this repository to your local machine using the following command:

```bash
git clone https://github.com/yourusername/JavaScript-Warm-Up.git
```

### Running the Scripts

Each script is named appropriately and can be executed as follows:

```bash
./script-name.js
```

Make sure to give execute permissions to the script using `chmod +x script-name.js` if necessary.

## Scripts

Here is a list of the scripts included in this repository:

1. **0-javascript_is_amazing.js**: Prints "JavaScript is amazing."

2. **1-multi_languages.js**: Prints three lines of text in different languages.

3. **2-arguments.js**: Prints a message depending on the number of arguments passed.

4. **3-value_argument.js**: Prints the first argument passed to it.

5. **4-concat.js**: Prints two arguments in a specific format.
